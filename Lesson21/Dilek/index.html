Form Validation in JS

JavaScript Form Validation için Kullanılabilen Property’ler

checkValidity()

Açıklama: Formdaki tüm inputların geçerli olup olmadığını kontrol eder. Eğer geçerli değilse, `false` döner. Bu metod, form gönderilmeden önce JavaScript ile çağrılır.
Kullanım:
const form = document.getElementById("myForm");
if (form.checkValidity()) {
// Form geçerli, gönderilebilir
} else {
// Form geçersiz, uyarı verebilirsiniz }

validity

Açıklama: Bu property, bir form elemanının geçerli olup olmadığını kontrol etmek için kullanılır.`validity` nesnesi, `true`/`false` dönen birçok alt property içerir.
`valid`: Alan geçerliyse `true`, değilse `false`.
`valueMissing`: Eğer bir alan boş bırakıldıysa `true` döner.
`typeMismatch`: Alanın tipi uyumsuzsa (örneğin, bir e-posta girilmesi gereken yere bir sayı yazıldıysa) `true` döner.
`tooShort`: Minimum uzunluk sağlanmadıysa `true` döner.
Kullanım:
const input = document.getElementById("email");
if (!input.validity.valid) {
if (input.validity.valueMissing) {
console.log("Bu alan zorunludur.");
}}

submit()

Açıklama: Formu JavaScript ile programlı bir şekilde gönderir. Ancak, bu metod çağrıldığında,
`submit` olayını tetiklemez ve dolayısıyla formun doğrulaması yapılmaz. Eğer doğrulama
yapılması isteniyorsa, `checkValidity()` metodunu kullanarak kontrol etmelisiniz.
Kullanım:
const form = document.getElementById("myForm");
form.submit(); // Formu gönderir
